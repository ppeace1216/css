CSS Animation property

Animation 속성 : animation-name(애니메이션 이름), animation-duration(애니메이션 실행시간), animation-timing-function(애니메이션 시간함수), animation-delay(애니메이션 지연시간), animation-iteration-count(애니메이션 실행횟수), animation-direction(애니메이션 흐름방향), animation-fill-mode(애니메이션 실행모드), animation-play-state(애니메이션 재생상태)



animation-name(애니메이션 이름)

애니메이션의 이름으로 필수 속성이며, 반드시 animation-duration과 @keyframes 규칙을 함께 지정해야 함

선택자 { animation-name:ani1; animation-duration:3s; }

@keyframes ani1 { 0% { top:0px; left:0px; } 100% { top:500px; left:600px; } }



animation 통합 속성

animation : name duration timing-function delay iteration-count direction fill-mode play-state

animation의 세부 속성을 한 번에 지정가능함



animation-duration, animation-name, @keyframes

animation-duration : 애니메이션의 실행시간을 나타내는 필수 속성으로 시간 단위로 값을 지정

animation-name : 애니메이션의 이름을 지정하는 필수 속성으로 영문과 숫자를 조합하여 지정

@keyframes : 애니메이션의 동작을 나타내는 키프레임들의 변화된 내용들을 기술하는 것으로 각 각의 키프레임은 %단위 또는 start, end로 지시한다.

선택자 { animation-name:애니메이션이름; animation-duration:5s; }

@keyframes 애니메이션이름 { start { top:0px; left:0px; } end { top:500px; left:600px; } }



애니메이션 선택 속성

animation-delay : 애니메이션의 지연 시간을 나타내는 선택 속성으로 시간 단위 값을 지정

animation-iteration-count : 애니메이션 실행 횟수를 나타내는 선택속성으로 반복횟수를 숫자로 지정 또는 무한대로 계속 실행하게 할 경우는 infinite를 적용

animation-direction : 애니메인션 실행방향을 지정하는 선택속성으로 0%에서 100% 방향을 순방향(), 100%에서 0% 방향을 역방향

animation-fill-mode : 애니메이션이 모두 진행된 후의 원래대로 복귀할지 아니면, 그대둘지 지정하는 옵션 - none, backwards, forwards, both

animation-timing-function : 애니메이션의 가속 및 감속 효과를 적용하는 선택속성으로 ease, ease-in, ease-out, ease-in-out, linear, cubic() 등을 선택하여 지정할 수 있음

animation-play-state : 애니메이션의 상태를 재생 또는 정지 상태로 변경할 수 있는 선택 속성으로 running, paused
